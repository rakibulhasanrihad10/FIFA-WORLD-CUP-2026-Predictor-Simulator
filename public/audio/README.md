# Celebration Audio Assets

You can place champion-specific audio files in this folder. When a team is crowned champion on the final screen, the simulator will automatically attempt to play its celebration theme track.

### File Naming Convention
Name the MP3 file using the lowercase 3-letter team ID:
- Argentina: `arg_winner.mp3`
- Brazil: `bra_winner.mp3`
- France: `fra_winner.mp3`
- Spain: `esp_winner.mp3`
- Portugal: `pt_winner.mp3`
- (And so on for any team: `[team_id]_winner.mp3`)

The files must be placed directly inside `/public/audio/`.
